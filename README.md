# 🎬 YouTube Insight Dashboard with Streamlit

An AI-powered YouTube Analytics Dashboard built using **Python, Streamlit, Plotly, and the YouTube Data API v3**. The dashboard provides interactive analytics, AI-powered insights, channel performance tracking, CSV analysis, and report generation.

## 🚀 Features

- 📊 Interactive Dashboard
- 🎥 Channel Analysis
- 📼 Video Analysis
- 📤 Upload CSV Dataset
- 🔑 YouTube Data API v3 Integration
- 🤖 AI Assistant
- 📈 Growth Prediction
- 🏆 Top 5 Videos Analysis
- 📊 AI Performance Score
- 🎯 Viral Video Prediction
- 🔥 Best Upload Time Recommendation
- 📄 AI Report Generation
- 📥 CSV & PDF Report Export
- 🌙 Dark / Light Theme
- 📊 Interactive Plotly Charts

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- YouTube Data API v3
- FPDF
- NumPy

## ▶️ Installation

```bash
git clone https://github.com/sonalirai254-lab/youtube-dashboard-with-streamlit.git

cd youtube-dashboard-with-streamlit

pip install -r requirements.txt

streamlit run streamlit_app/app.py
```
